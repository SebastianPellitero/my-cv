# sebastianpellitero.dev

Personal portfolio site built to showcase my work, stack, and approach as a Senior Frontend Developer.

## Stack

- **Next.js** (App Router)
- **TypeScript**
- **Tailwind CSS**
- Deployed on **Vercel**

## What's in it

A minimal, fast-loading site focused on three things I care about most as a developer: frontend architecture, performance, and accessibility. No bloat, no templates — built from scratch.

## Running locally
```bash
npm install
npm run dev
```
